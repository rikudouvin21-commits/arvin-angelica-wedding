# 💍 Arvin & Angelica Wedding Website

A modern luxury digital wedding invitation built with **Next.js**, **React**, **TypeScript**, **Tailwind CSS**, and **Framer Motion**.

Live Website:

> https://arvin-angelica-wedding-v2.vercel.app

---

## ✨ Features

- Luxury Hero Section
- Our Story
- Relationship Timeline
- Wedding Venue Details
- Countdown Timer
- Editorial Gallery
- Wedding Day Schedule
- Elegant Footer

Upcoming Features

- RSVP System
- Guest Dashboard
- Gallery Lightbox
- Interactive Maps
- Calendar Download

- Floral Theme

## 🚀 Development

Install dependencies

```bash
npm install
```

Run locally

```bash
npm run dev
```

Build

```bash
npm run build
```

---

## 🌐 Deployment

This project is deployed using **Vercel**.

Every push to the `main` branch automatically deploys the latest version.
